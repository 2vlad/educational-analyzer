Введение в JavaScript

JavaScript - это язык программирования, который используется для создания интерактивных веб-страниц. Давайте изучим основы.

Переменные
В JavaScript есть три способа объявления переменных:
- var (старый способ)
- let (современный способ) 
- const (для констант)

Пример:
let name = "Иван";
const age = 25;

Функции
Функции позволяют группировать код для повторного использования.

function greet(name) {
  return "Привет, " + name;
}

Циклы
Циклы используются для повторения кода.

for (let i = 0; i < 10; i++) {
  console.log(i);
}

Заключение
JavaScript - мощный и популярный язык программирования.
